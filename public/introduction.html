🎯 개발 배경

🔓

무분별한 AI 사용의 문제점

현재 학생들은 ChatGPT나 Gemini 같은 AI를 자유롭게 사용하지만, 교육적 목적에 부합하지 않는 결과물을 얻거나 부적절한 내용에 노출될 위험이 있습니다.

👩‍🏫

교사 통제의 필요성

교사가 AI 사용을 관리하고 모니터링할 수 있는 환경이 필요합니다. 학습 목표에 맞는 AI 활용과 학생의 학습 과정을 체계적으로 추적할 수 있어야 합니다.

📊

개별 맞춤 교육의 어려움

전통적인 일률적 교육으로는 각 학생의 수준과 특성을 고려한 개별 지도가 어렵습니다. AI를 활용한 맞춤형 교육 시스템이 필요합니다.

💡

교육 과정 투명성 확보

학생들이 어떻게 학습하고 있는지, 어떤 어려움을 겪고 있는지 실시간으로 파악할 수 있는 도구가 필요합니다. 데이터 기반의 객관적인 분석이 중요합니다.

🚀

미래 교육 역량 강화

AI 시대에 맞는 교육 방법론이 필요합니다. 학생들이 AI와 협력하는 방법을 배우고, 교사는 AI를 효과적으로 활용한 교육을 실현해야 합니다.

🔐

안전하고 통제된 학습 환경

교사의 승인과 관리 하에서만 AI를 사용할 수 있는 안전한 환경을 제공하여, 학생들의 건전한 AI 학습 경험을 보장합니다.

📖 교육학적 이론 배경

🏛️ 이론적 토대

본 플랫폼의 프롬프트는 검증된 교육학 이론들을 기반으로 설계되어,
단순한 정답 제공이 아닌 학생의 사고 과정을 체계적으로 발달시키는 데 중점을 둡니다.

💭

소크라테스식 교수법

질문과 대화를 통한 학습: AI 튜터는 답을 직접 제시하기보다 적절한 질문을 통해 학생 스스로 사고하고 답을 찾아가도록 안내합니다. 이는 비판적 사고력과 메타인지 능력을 기릅니다.

🎯

Bloom의 교육목표 분류학

체계적 학습 단계: 기억 → 이해 → 적용 → 분석 → 종합 → 평가의 6단계를 따라 학생의 인지적 발달을 단계적으로 지원합니다. 각 프롬프트는 학생의 현재 단계를 파악하고 다음 단계로 나아가도록 설계됩니다.

🏗️

비고츠키의 근접발달영역

적정 수준의 도전: 학생이 혼자서 할 수 있는 것과 도움을 받아서 할 수 있는 것 사이의 영역에서 학습이 이루어지도록 합니다. AI 튜터는 적절한 비계(scaffolding)를 제공합니다.

🔄

구성주의 학습이론

능동적 지식 구성: 학생이 기존 지식과 새로운 정보를 연결하여 스스로 의미를 만들어가도록 돕습니다. AI는 정보 제공자가 아닌 학습 촉진자 역할을 합니다.

🎨

다중지능이론

다양한 학습 스타일 지원: 언어, 논리수학, 공간, 음악, 신체운동, 인간친화, 자기성찰, 자연친화 지능을 고려하여 각 학생의 강점을 활용한 맞춤형 접근을 제공합니다.

📈

형성평가 이론

지속적 피드백: 학습 과정에서 실시간으로 학생의 이해도를 파악하고 즉시 피드백을 제공합니다. 이를 통해 학습 격차를 최소화하고 개별 맞춤 학습을 실현합니다.

🔬 과목별 특화 이론

📖 국어: 독자반응이론

학생의 개인적 경험과 배경지식을 바탕으로 텍스트를 해석하고 의미를 구성하도록 지원

🔢 수학: 문제해결이론

Polya의 4단계(이해-계획-실행-검증) 문제해결 과정을 통한 체계적 수학적 사고력 개발

🏛️ 사회: 비판적사고이론

다양한 관점에서 사회 현상을 분석하고 논리적 근거를 바탕으로 판단하는 능력 배양

🔬 과학: 탐구학습이론

관찰-가설-실험-결론의 과학적 탐구 과정을 통한 과학적 사고력과 문제해결능력 신장

✨ 주요 기능

🎯

과목별 맞춤 AI 튜터

국어, 수학, 사회, 과학 각 과목의 특성에 맞는 AI 튜터가 학생들의 학습을 도와줍니다. 교육학적 이론에 기반한 체계적인 피드백을 제공합니다.

📊

수업 관리 시스템

QR코드를 통한 간편한 수업 참여, 실시간 학생 참여 현황 모니터링, 수업별 체계적인 데이터 수집이 가능합니다.

🧠

학습 분석 도구

학생들의 대화 내용을 분석하여 학습 수준, 참여도, 사고 패턴을 파악하고 개별 맞춤 학습 방향을 제시합니다.

🎨

맞춤형 프롬프트 설정

교사가 직접 AI 튜터의 대화 스타일과 교육 방향을 설정할 수 있어, 수업 목표에 맞는 개성 있는 AI 튜터를 만들 수 있습니다.

📱

모바일 최적화

스마트폰, 태블릿, 데스크톱 등 모든 기기에서 원활하게 작동하여 언제 어디서나 학습이 가능합니다.

🔒

안전한 학습 환경

교사 승인 하에만 접근 가능한 통제된 학습 환경으로, 학생들의 안전한 AI 학습 경험을 보장합니다.

📋 사용 방법

1

교사 로그인

구글 계정으로 간편하게 로그인하고 API 키를 설정합니다.

2

수업 생성

새 수업을 생성하고 과목을 선택한 후 QR코드를 생성합니다.

3

학생 참여

학생들이 QR코드를 스캔하여 AI 튜터와 대화를 시작합니다.

4

실시간 모니터링

교사 대시보드에서 학생들의 학습 현황을 실시간으로 확인합니다.

5

학습 분석

AI 분석 결과를 통해 각 학생의 학습 상태를 파악하고 피드백합니다.

🎓 교육적 효과

🎯 개별 맞춤 학습

각 학생의 수준과 특성에 맞는 개별화된 학습 경험을 제공합니다.

💬 소크라테스식 대화

질문과 대화를 통한 능동적 학습으로 사고력을 키워줍니다.

📊 데이터 기반 교육

객관적인 학습 데이터를 바탕으로 한 과학적 교육 방법을 제공합니다.

⏰ 24시간 학습 지원

언제든지 접근 가능한 AI 튜터로 지속적인 학습을 지원합니다.

🔍 학습 과정 가시화

학습 과정과 사고 패턴을 시각적으로 분석하여 학습 효과를 극대화합니다.

🎪 흥미로운 학습

AI와의 대화형 학습으로 학생들의 참여도와 흥미를 높입니다.

🏛️

🔬

과학

🤖 AI 기술

📚

1단계: 지식 관리자

과목별 JSON 설정 파일을 토대로 학습 규칙과 데이터를 체계적으로 관리하고, 캐시 기능으로 빠른 응답을 보장합니다.

🧐

2단계: 사고 분석가

학생의 메시지를 다각도로 분석하여 학습 상태와 의도를 파악하고, 대화 맥락과 학습 진행 과정을 종합적으로 진단합니다.

✍️

3단계: AI 프롬프트 설계자

분석 결과를 바탕으로 교육학적 이론과 과목별 특성을 반영한 최적의 AI 지시사항을 동적으로 조합하여 생성합니다.
